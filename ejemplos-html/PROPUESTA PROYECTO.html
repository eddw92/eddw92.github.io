MI PROPUESTA DE PROYECTO

RP PARA DENTISTAS

CORPUS SANCHEZ JOSE EDUARDO

Como propuesta de proyecto se tiene pensado hacer una base de datos para almacenar informacion acerca de las citas de algunos dentistas, ya que a veces como clientes tenemos problemas con el horario, algunos imprevistos, un dentista conocido no esta disponible, asi que se desea crear una pagina web la cual pueda tener el control de todo lo que sea necesario para que sea mas facil el poder recordar citas, recervaciones, consulta de informacion, etc.

Aun no sabemos mucho de como se quiere planear esto, pero al menos que sea de una forma muy didactica y que la pagina web sea amigable a cualquier tipo de persona ya que se espera que tanto jovenes como adultos tengan acceso a informacion necesaria para solicitar un servicio, asi como el manejo por parte de los mismos dentistas, ya que ellos tendran un acceso especial para manejar sus actividades como empleados.

Se piensa utilizar imagenes, videos, animaciones para poder hacer que la pagina sea aun mas agradable. Se desea tener comentarios de personas que han tenido experiencias con dentistas a la que se enfoca la pagina web, en cuanto al servicio, lo que necesitan mejorar, o sus agradecimientos.

Como una actividad extra se tiene el que se use una aplicacion para poder llevar todo esto a un telefono movil en caso de que los clientes no cuenten a la mano con un ordenador.
